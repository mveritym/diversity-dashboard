Diversity Dashboard makes visualizing your company's gender breakdown as easy as dragging and dropping a file. Input files are never saved so your information is always kept private.

Uploading A File

Your input file should be a CSV file with one employee's information per line. You'll need a header row with at least the following titles: "Gender" and "Role" (not case-sensitive). All extra columns will be ignored.

For example, in this data set the employee ID and name columns will be ignored. The input file will not be saved so this information stays private.

If you have an Excel file of employee data you can export it to a CSV file easily using these instructions.

When you have a file ready, just drag and drop your file onto the page or click in the white box to upload your file.
